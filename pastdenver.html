This is part of my independent study for the Spring semester of 2015. The whole project consisted in explore how web mapping could be used to visualize spatio-temporal data in a interactive environment. In addition, this map reflects an attempt to create a solid framework to manipulate spatial data on the web using database systems (PostgreSQL) and web geo services (Geoserver). However, note that the map that you will explore here do not use such technologies given the fact that this framework requires a more flexible hosting environment. In addition to this map, I also developed the São Paulo’s Residential Development web map to explore other angles of spatio-temporal data.
